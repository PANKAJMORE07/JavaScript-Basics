What is JavaScript ?
1. JavaScript is programming language used to make websites interactive and dynamic
2. it works alongside HTML and CSS to add behaviour to web pages
3. popularly known as the language of the web

Where does js run?
1. in the browser - built in engine executes javascript(e.g v8 engine in chrome)
2. on servers - with node.js - provides environment to run js ie. it has v8 engine
3. the v8 engine by Google made js significantly faster and more efficient

What can js do?
1. update content dynamically
2. responds to user actions like click, typing etc.
3. perform animations like slideshows, fading effects etc
4. communicate with servers to load/save data eg. APIs


Js Pros:
1. Versatile: use for both frontend and backend
2. Popular: Massive global community
3. Fast Execution: code run directly in the browser
4. Asynchronous: supports event driven & non blocking operations eg async await 
5. Cross platform: can be used for web, mobile & deskstop apps
6. Rich ecosystem: extensive libraries & framworks speed up development

Js Cons:
1. Weekly Typed: we can change the type of variables
2. Browser Compatability: may not work uniformaly across all browser
3. Single-threaded: executes single task at a time
4. Debugging Challenges: async code, dynamic typing can make drbugging difficult
